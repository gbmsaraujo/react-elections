# React Elections
Desafio do módulo 02 do Bootcamp em React oferecido pela XP Educação.

# Tecnologias Utilizadas

`- React
 - TailWind
 - Node.js
`
# Como rodar o front? 

- Ter o node na versão 14;
- Npm install ou yarn.

# Como rodar o front? 

- Acessar a pasta backend pelo terminal;
- Rodar um yarn;
- Depois yarn serve.
